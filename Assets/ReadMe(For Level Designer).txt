

-> To add a new level; Drag a "GameObject" that must be named "Level" to the "Level List" list of the "GameManager" script.

-> To create a new level, you can use object in this path "Prefabs-> Empty Level Template-> Level".

-> While designing the level, basic shapes from "Objects" folder can be used. Positions and scales of the objects are OK for platform.


-> It is important that the name of the Level object is "Level", the first child is "First Stage",
	the second child is "Mid Stage",
	the third child is "Second Stage" and the relevant tags should have been assigned.
